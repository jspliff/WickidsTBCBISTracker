# Wick's TBC BIS Tracker

> Best-in-slot gear tracking for every class, every spec, every phase of TBC Classic.

Part of the **[Wick suite](../WickSuite)** — three precision addons built around a single fel-green-on-deep-purple aesthetic.

## Features

- **Full BIS lists** for every class and spec, across all TBC phases (P1–P5).
- **Acquisition tracking** per slot — mark as obtained, ignored, or still hunting.
- **Item tooltip preview** on hover; click to mark.
- **SoftRes integration** — import/export roster.
- **Phase-aware** — flip phases without losing state.
- **Custom lists per character**, shared settings account-wide.

## Install

- **CurseForge:** search for "Wick's TBC BIS Tracker".
- **Manual:** drop the `WickidsTBCBISTracker` folder into `World of Warcraft\_classic_\Interface\AddOns\`.

## Usage

```
/wtbt
```

Opens the panel. Pick class → spec → phase. Start ticking items.

## Compatibility

- **TBC Classic** (Burning Crusade / Anniversary) — Interface `20505`.
