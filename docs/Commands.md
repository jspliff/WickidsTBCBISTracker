# Slash Commands

| Command | Effect |
|---|---|
| `/wtbt` | Toggle the main panel |
| `/wtbt show` | Open the panel |
| `/wtbt hide` | Close the panel |
| `/wtbt reset` | Reset panel position to center of screen |

## In-panel actions

- **Class / Spec / Phase dropdowns** — switch BIS list.
- **Click item row** — mark acquired / hunting.
- **Shift-click item row** — toggle ignore (strikethrough).
- **Hover** — item tooltip.
- **Drag header** — move panel.
- **Drag corner** — resize.

## SoftRes

- **Paste token** into the import box → your raid's reservations highlight matching items.
- **Export** writes a token string you can paste back into softres.it.
