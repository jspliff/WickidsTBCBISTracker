# Getting Started

## Install

- **CurseForge client** — search for "Wick's TBC BIS Tracker" and install.
- **Manual** — download the latest ZIP from [CurseForge](https://www.curseforge.com/wow/addons/wicks-tbc-bis-tracker) or [GitHub Releases](https://github.com/jspliff/WickidsTBCBISTracker/releases), extract the `WickidsTBCBISTracker` folder into `World of Warcraft\_classic_\Interface\AddOns\`.

## First run

1. Log in to a character.
2. `/wtbt` — opens the panel.
3. Pick your **class** (top selector) and **spec** (next dropdown).
4. Pick a **phase** — P1 through P5 are supported.
5. The BIS list loads. Hover any item for the tooltip; click to mark acquired.

## What the markers mean

- **Green border / filled** — item acquired.
- **Dim** — item still hunting.
- **Strikethrough** — item ignored (you've decided you don't want it).

## SoftRes integration

Paste a SoftRes token string into the import field and the panel will highlight items reserved by your raid.
